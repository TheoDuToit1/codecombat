X-Code Crew Academy: Gauntlet II Integration Overview

🌟 Key Gauntlet II Features Implemented
1.	Health Depletion System
o	Health decreases gradually over time.
o	Reinforces the importance of resource management and code efficiency.
o	Players must find food (represented as health pickups) to survive.
2.	Monster Generators
o	Spawn enemies continuously until destroyed.
o	Teach infinite loops and interrupt logic.
3.	Four Character Classes
o	Warrior: Strength-based; excels in direct combat.
o	Valkyrie: Balanced defense/offense; organizational logic.
o	Wizard: Master of logic and spells; high-damage abilities.
o	Elf: Speed and precision; evasion and pathfinding.
o	Each class now teaches a distinct coding approach.
4.	Voice Announcements
o	Classic lines like "Warrior needs food, badly!"
o	Integrated with HUD alerts and logic prompts.
5.	Treasure Collection
o	Food = Health
o	Potions = Code power-ups (e.g. reusable functions)
o	Keys = Unlock conditional logic paths
o	Gold = Score-based achievements
6.	Special Abilities
o	Each class has one unique special:
	Warrior: Area burst (function loops)
	Valkyrie: Shield (error catch blocks)
	Wizard: Fireball (condition chains)
	Elf: Dash (recursive shortcuts)
7.	Hazards & Obstacles
o	Fire, acid puddles, spike tiles.
o	Force learners to reroute and plan ahead.
8.	"It" Tag System
o	Special enemy that spreads an error condition.
o	Represents bugs that cascade through poorly isolated code.
9.	Gauntlet-Style HUD
o	Health, Food, Score, and Time.
o	All dynamically update based on in-game logic.
10.	Boss Battles
o	End-of-level strategic fights.
o	Teach modular code reuse, timing, and conditional flow.
